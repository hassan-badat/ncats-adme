HLM Stability

Name: Human Liver Microsomal Stability
Dataset Type: Classification
Dataset Size: 6648 compounds
Validation: 5-Fold CV
Descriptors: RDKit descriptors
Model Details: The best model is based on the XGBoost algorithm. The XGBoost model comprises of 300 trees and uses a selected list of RDKit descriptors as features. The models were built using scikit-learn library.
Interpretation: The model provides predicted class (1 or 0) for a given compound. Data Availability: A subset of the dataset used for developing these models will be deposited to the PubChem Bioassay database.

HLM Assay Protocol

The substrate depletion method, also called the in vitro t1/2 method was used to analyze test compounds. Each reaction mixture (110 μL) consisted of a test article (1 μM), human liver microsomes (HLM) (0.5 mg/mL), and NADPH regenerating system in phosphate buffer at pH 7.4. Samples were incubated in 384-well plates at 37°C for 0, 5, 10, 15, 30 and 60 min.

Additional Models

An additional HLM stability model was trained on three datasets from Biogen, ChEMBL, and NCATS — each having 3,068, 6,092, and 7,509 compounds respectively. This XGBoost model is wrapped with a Balanced Bagging Classifier and uses RDKit molecular descriptors. The model can be accessed on GitHub at ncats-adme/additional_models/hlm_stability.

1. HLC Stability

Name: Human Liver Cytosolic Stability
Dataset Type: Classification

To address the class imbalance (14.2% unstable compounds), a Stratified Bagging ensemble strategy was implemented. The models were implemented using TensorFlow and Keras.

2. MLC Stability

Name: Mouse Liver Cytosolic Stability
Dataset Type: Classification
Dataset Size: 2,856 compounds
Validation: 5-Fold Stratified CV & Internal validation
Descriptors: Consensus (Fingerprints (Morgan, Avalon, AtomPair, and RDKit) + 2D descriptors (Mordred and RDKit))
Model Details: The final model aggregates predictions from 4 Deep Neural Network (DNN) models trained on balanced subsets. The DNN architecture features two hidden layers (256 and 128 neurons) with ReLU activation and dropout (rate = 0.3). The model was trained using the Adam optimizer for 10 epochs with a batch size of 64. To mitigate severe class imbalance (8.5% unstable compounds), a Stratified Bagging ensemble strategy was applied. The models were implemented using TensorFlow and Keras.

3. RLC Stability

Name: Rat Liver Cytosolic Stability
Dataset Type: Classification
Dataset Size: 1,198 compounds
Validation: 5-Fold Stratified CV & Internal validation
Descriptors: RDKit Descriptors
Model Details: Due to extreme class scarcity (4.9% unstable compounds), the model utilizes a Stratified Bagging ensemble of 64 Random Forest (RF) models. Each RF estimator is configured with 100 trees, using the square root of features for splitting. The model was built using the scikit-learn library and RDKit molecular descriptors.

Inference

The model predicts stability class (1: Unstable, t1/2 ≤ 30 min; 0: Stable, t1/2 > 30 min) and a probability score (between 0 and 1). An Applicability Domain (AD) threshold of 0.7 (Maximum Tanimoto similarity score) is used to flag low-confidence predictions.

Data Availability

The datasets and models are available through the NCATS@ADME platform.

Assay Protocol

Mixed gender human (H0610.C), male CD-1 mouse (M1000.C), and male Sprague Dawley rat (R1000.C) liver cytosol fractions were purchased from Sekisui XenoTech, LLC (Kansas City, USA). The substrate depletion method was used to determine in vitro t1/2. The assay (384-well format) involved incubating a test compound (1 µM) with cytosol fractions (2 mg/mL) in phosphate buffer (100 mM, pH 7.4) at 37 °C. Samples were collected at 0, 5, 10, 15, 30, and 60 min. Percent remaining of the parent compound was analyzed using an integrated LC/MS method, and t1/2 was calculated as described in previous work (Shah et al., 2016; Shah et al., 2020). Compounds with t1/2 > 30 min were classified as stable, while those with t1/2 ≤ 30 min were classified as unstable.
